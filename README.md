# 🚀 TranGPT · 聚合聊天 + AI 翻译平台

一个后台，管理所有客户对话 💬
多平台统一收件箱 + 实时翻译 + 自动化运营工具

## ✨ 项目简介

TranGPT 是一款面向跨境业务 / 客服 / 营销团队的 全平台聊天聚合系统，支持多个社交平台统一管理，并内置 AI 翻译与自动化能力。

不再来回切换软件，一站式完成沟通、翻译、客户管理与运营 🚀

### 🌍 核心功能
💬 多平台聊天聚合
支持 WhatsApp / Telegram / LINE / Facebook / Zalo
所有消息统一收件箱
多账号集中管理
支持团队协作（客服分配 / 标签）

### 🌐 实时 AI 翻译
收到消息自动翻译
发送消息自动转为对方语言
支持多语言（中 / 英 / 东南亚语言）
接入 LLM（可自定义模型）

### 🤖 自动化与智能助手
自动回复（关键词 / AI）
客户标签 & 分组管理
对话摘要（AI总结）
智能推荐回复

### 📊 客户与运营管理（SCRM）
客户画像（来源 / 行为 / 标签）
会话历史记录
转化跟踪
数据统计分析
