# 🚀 TranGPT · 聚合聊天 + AI 翻译平台

一个后台，管理全球客户沟通 🌍
多平台收件箱 + 实时 AI 翻译 + 自动化运营工具

## ✨ 项目简介

TranGPT 是一款面向 跨境业务 / 客服团队 / 营销增长团队 的
全平台智能聊天聚合系统（Chat Aggregation + AI Translation + SCRM）

帮助企业解决：

❌ 多平台来回切换

❌ 多语言沟通困难

❌ 客户数据分散

❌ 客服效率低

### 👉 通过一个系统全部解决：

💬 沟通 + 🌐 翻译 + 📊 客户管理 + 🤖 自动化运营

## 🌍 核心功能
💬 多平台聊天聚合（Unified Inbox）
支持平台：WhatsApp / Telegram / LINE / Facebook / Zalo等

所有消息统一收件箱

多账号集中管理

团队协作：客服分配，会话接管，标签体系

👉 告别切换软件，一个界面处理所有客户

## 🌐 实时 AI 翻译（AI Translation Engine）
收到消息 👉 自动翻译

发送消息 👉 自动转为对方语言

支持语言：
中文 / 英文 / 东南亚语言（泰语 / 越南语 / 印尼语等）

支持接入：
自定义 LLM（本地 / OpenAI / 私有模型）

👉 像用母语一样做全球生意 🌍

## 🤖 自动化与智能助手（Automation + AI Agent）
自动回复：
关键词规则
AI 智能回复

AI 能力：
对话摘要
智能推荐回复
客户意图识别（可扩展）
自动标签 / 分组

👉 7x24 小时自动接待客户，提高转化率

## 📊 客户与运营管理（SCRM）
客户画像：

来源渠道

行为记录
